Jquery Listbox plugin

By Droc101

Methods

Method Purpose Args
init(); Initiates the plugin and loads css.  
create(id, location); Creates a listbox id: the html id of the listbox, location: what element to append the listbox to
add(text, id, box); Add an item to a listbox text: the item text, id: the html id of the item, box: the listbox to add the item to
addHandler(code, id); Add an event handler to an item. code: the javascript code to execute on click, id: the html id of the item to attach to

Requirments

  • Jquery 3.3.1
  • CSS3 Support
